ИнтернетПрокси / InternetProxy

Параметры прокси-сервера для доступа в Интернет. В текущей реализации поддерживается только HTTP прокси. Стандартные методы объекта ИнтернетПрокси из 1С:Предприятие для FTP и SOCKS не реализованы.
